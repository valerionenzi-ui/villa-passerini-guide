# Relais Villa Passerini · Guida Ospiti

Guida digitale interattiva per gli ospiti del Relais Villa Passerini, agriturismo a Roselle (Grosseto), Maremma Toscana.

Dashboard mobile-first bilingue IT/EN: informazioni pratiche, regole, suggerimenti sul territorio, contatti.

## Sviluppo locale

```bash
npm install
npm run dev
```

## Build per produzione

```bash
npm run build
```

I file ottimizzati vengono generati nella cartella `dist/`.

## Deploy

Hosting consigliato: Vercel (deploy automatico da GitHub).

## Stack

- React 18
- Vite
- Foto ospitate su repository GitHub separato (`villa-passerini-assets`)
